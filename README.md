# Fobinacci-clock

Fobinacci-clock ist eine Desktop-Anwendung (Fibonacci-Clock), die die Uhrzeit mithilfe der Fibonacci-Farb-/Blockdarstellung visualisiert.

Kurzbeschreibung
- Zeigt die Uhrzeit in einem visuellen Raster, basierend auf der Fibonacci-Folge.
- Implementiert als .NET Framework 4.8 Projekt; Benutzeroberfläche mit Windows Forms / WPF (je nach Projektdatei).

Voraussetzungen
- Windows mit installiertem .NET Framework 4.8
- Visual Studio (Empfohlen: Visual Studio 2022/2026) zum Bauen und Entwickeln

Build & Ausführung
1. Repository klonen oder lokal öffnen.
2. Die Lösung in Visual Studio öffnen (.sln-Datei).
3. Zielkonfiguration wählen (Debug/Release) und die Lösung bauen (Strg+Shift+B).
4. Projekt als Startprojekt markieren und ausführen (F5) oder die erzeugte EXE im Ausgabeordner starten.
